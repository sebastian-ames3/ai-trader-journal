# Project Overview

## Product
**AI Trader Journal** - Mobile-first trading psychology journal with AI-powered behavioral analysis.

## Stack
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Prisma + PostgreSQL (Supabase)
- shadcn/ui
- Claude (AI analysis) + Whisper (voice transcription)

## Core Value Proposition
Help traders:
1. Capture journal entries with minimal friction (voice, text, screenshots)
2. Receive AI-powered insights on trading psychology and patterns
3. Track theses and trades with P/L analysis
4. Improve through AI coaching and accountability features

## Current State
- Core application complete (Phases 1-4)
- PWA basics complete, native deployment paused
- Active work: PRD 1 - Digitization & Import (Step 2 of 5)

## Key Personas
1. **Active Trader** - Logs trades, wants pattern analysis
2. **Journal Writer** - Focuses on psychology and reflections
3. **Mentor/Mentee** - Uses sharing and accountability features

## Success Metrics
- User can capture entry in <30 seconds
- AI analysis adds actionable insight
- Weekly insights surface behavioral patterns
- Trade tracking provides accurate P/L
